# OpenGL Setup Script

### Requirements

* x64 Windows PC
* Code::Blocks with MinGW compiler installed

### Usage

1. Open **PowerShell as Administrator**.
2. Run the setup script.
3. Open **Code::Blocks**.
4. Select **Open an existing project**.
5. Open:

   ```
   Downloads\qwerty\qwerty.cbp
   ```
6. Build and Run.

Enjoy!
